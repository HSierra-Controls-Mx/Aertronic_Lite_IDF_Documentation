The Sensors.h file contains functions declarations for the analog sensors used in the AERtronic Lite project.

sensors_init

esp_err_t sensors_init(void)

Initialize the sensors. This function sets up the ADC channels for reading pressure and temperature sensors. It should be called before any sensor readings are performed.

Author

Ernesto Estrada

Date

June 10, 2025

Version

1.0

Returns:

ESP_OK on success, or an error code on failure.

Declaration

esp_err_t sensors_init(void) {
    // Initialize ADC configuration for unit 1
    adc_oneshot_unit_init_cfg_t init_config = {
        .unit_id = ADC_UNIT_1, // Use ADC unit 1
        .clk_src = ADC_RTC_CLK_SRC_DEFAULT, // Use default clock source
    };

    if(sensor_data_mutex == NULL) {
        sensor_data_mutex = xSemaphoreCreateMutex(); // Create a mutex for sensor data access
    }

    // Create a new ADC unit with the specified configuration and handle
    ESP_ERROR_CHECK(adc_oneshot_new_unit(&init_config, &adc1_handle));

    // Configure the ADC channels for pressure and temperature sensors
    adc_oneshot_chan_cfg_t chan_config = {
        .atten = ADC_ATTEN,
        .bitwidth = ADC_WIDTH,
    };
