esp_err_t sensors_read ( sensor_data_t * data)

Read sensor data. This function reads the current values from the pressure and temperature sensors. It populates the provided sensor_data_t structure with the latest readings.

Parameters
dataPointer to a sensor_data_t structure where the sensor readings will be stored.
Returns
ESP_OK on success, or an error code on failure.
Author
Ernesto Estrada
Date
June 10, 2025
Version
1.0
Note
The function assumes that sensors_init() has been called successfully before this function. It will read the ADC values, convert them to pressure and temperature, and store them in the provided structure.
