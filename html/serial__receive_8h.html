◆ serial_read()

void serial_read ( void )

Function to read data coming from the serial port. This function is design to handle the data from the display and process it accordingly.

Note
The command received has the following format: code + data1 + data2.
  • The code is a single byte that identifies the command.
  • The data1 is a single byte that contains the value associated with the command.
  • The data2 is a single byte that contains additional information related to the command. Defaults to 0x00 if not used.

This function does not return any value.
