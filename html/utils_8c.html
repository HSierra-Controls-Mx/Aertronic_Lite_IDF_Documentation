void get_status_attributes ( int32_t state,
int status_attributes[4] )

Updates the header to reflect the current system state. This function updates the header on the Nextion display to show the current system state.

Parameters
system_stateThe current state of the system (0 = Normal, 1 = Warning, 2 = Alarm).
status_attributesAn array containing status attributes to be displayed in the header. The array should contain 6 elements representing different status indicators.

This function does not return any value.

◆ get_status_text()

const char * get_status_text ( int32_t state,
int32_t language )

Retrieves the status text based on the current system state and language. This function returns a string representing the status text for the given state and language.

Parameters
stateThe current state of the system (0 = Normal, 1 = Warning, 2 = Alarm).
languageThe language code (0 = English, 1 = Spanish, 2 = Portuguese, 3 = French, 4 = German).
Returns
A string containing the status text in the specified language.

The function uses a switch-case structure to determine the appropriate status text based on the state and language.
