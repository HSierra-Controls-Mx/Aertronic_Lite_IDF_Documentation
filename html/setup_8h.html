void display_ready ( void )

Waits for the Nextion display to send the command indicating that it is ready.

This function blocks until the display sends the command: 0x88, 0xFF, 0xFF, 0xFF

Note
This function should be called right after the initialization of the UART and NVS.

◆ wait_for_datetime()

void wait_for_datetime ( void )

Blocks until a valid date/time string is received via UART.

The expected string format is p00,MM,HH,DD,MM,YYYY followed by three 0xFF bytes.

Warning
Ensure that the Nextion display is properly configured to send the date and time in the expected format.
Note
This function should be called before starting any FreeRTOS tasks.
