◆ about_page()

void about_page ( void )

Function to handle the about page UI. This function is called when the current page is set to ABOUT.

This function should contain the logic to update the about page of the Nextion display. It may include updating text fields, images, and other UI elements specific to the about page. This function does not return any value.

Note
this function is part of the ui_task and is called when the task is running and the current page is set to ABOUT.
